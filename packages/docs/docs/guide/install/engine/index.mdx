Chronocat 自身实现了 Satori 协议，但并不实现任何特定的功能；这些功能由 Chronocat
引擎提供。你可以为 Chronocat 安装引擎来添加功能。

## :star: 一句话说明 :star:

**绝大多数情况下，安装「chronocat-api」和「chronocat-event」两个引擎即可。** 在 Chronocat
下载页即可找到两个引擎的插件。
